# AI Blog Writer Agent

- An autonomous, multi-agent content generation platform that leverages LangGraph and Generative AI to produce structured, professional technical articles. This project demonstrates a sophisticated "Plan-and-Execute" workflow, complete with automated research, multi-section drafting, and AI-driven technical illustration.

- Hosting: Containerized with Docker and deployed on Hugging Face Spaces.

- Frontend: Built with React.js, featuring a real-time Markdown preview and PDF export capabilities.

# 🧠 The Agentic Pipeline

This project utilizes a Map-Reduce architectural pattern managed by LangGraph to ensure logical flow and high-quality output:

## 1. The Planner (Strategic Architect)
- Purpose: Content structuring and outline generation.

- How it works: This node takes the initial blog title/topic and acts as a Senior Editor. It generates a comprehensive Plan that includes optimized headings, specific word counts for each section, and a defined audience tone. This ensures the blog has a logical flow before a single paragraph is written.

## 2. The Writer (Domain Expert)
- Purpose: Technical drafting and content generation.

- How it works: Using a Fan-out pattern, the Writer node triggers parallel tasks for every heading generated by the Planner. It treats each section as an individual drafting assignment, ensuring high technical depth and covering all bullet points without losing focus on the main topic.

## 3. The Assembler (Synthesis & Visual Integration)
- Purpose: Final compilation and multi-modal enhancement.

- How it works: This node performs the Fan-in (merging) of all written paragraphs into a final Markdown document. Simultaneously, it analyzes the content to identify logical "visual breaks." It generates detailed image prompts, calls the image generation API, and dynamically embeds the resulting diagrams/illustrations into the final blog post.

# 🛠️ Technical Stack

## Backend

- Orchestration: LangGraph (Stateful Multi-Agent Workflows)

- Framework: FastAPI (Python)

- Models: Llama-3.3-70b (via Groq), FLUX.1-schnell (Image Gen)

- Logic: Pydantic (Data Validation), OpenAI/LangChain Schemas

- Environment: Docker

## Frontend

- Framework: React.js

- Markdown: React-Markdown with Syntax Highlighting (Prism)

- Export: Custom HTML-to-PDF conversion logic

- Styling: Tailwind-inspired CSS-in-JS with high-performance animations
